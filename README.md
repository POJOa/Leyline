# src.moe
用于实现萌豚友链互换和轻量级讨论版的Py交易网站。
基于master分支的Leyline。

# Usage

Launching
``````
mvn clean install

mvn spring-boot:run
``````

Development

Framework should not be touched fluently.

Try to implement in interfaces and business packages.

# Environment

Spring Boot / Spring Security / Spring Data JPA + QueryDSL + Hibernate 5

Angular 1.x(可能)

Redisson

MySQL
